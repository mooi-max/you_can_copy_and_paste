## 💡 简介 (About)

🎯 **专门拯救“网页禁止复制粘贴”的效率神器。**

在学习通、头歌等平台做作业时，明明有现成的文字，却因为网页限制只能苦哈哈地纯手工打字？

面对这种无理设定，**You-Can-Copy-And-Paste** 给出了最解气的答案：既然你不让我“贴”，那我就让电脑帮我“打”！这款超小巧的绿色软件，能无视所有网页限制，把你复制的内容瞬间变成“模拟键盘打字”输入进去。不用任何复杂设置，双击即用，把本该属于你的时间省下来！彻底绕过网页的拦截规则，让你在任何地方都能畅快地复制粘贴！

## 📝 进阶技巧：对抗终极防御 (Advanced Tips)

遇到连强行复制都失效、甚至鼠标根本**无法选中文本**的“终极防御”网页（例如使用了深度 Canvas 渲染或严格 CSP 拦截的站点），请使用以下**降维打击**方案获取代码。

⚠️ **核心思路**：无论你用哪种方式把文本“扒”进了剪贴板，最后面对平台死皮赖脸的**“禁止粘贴”**时，都请使用本工具的**粘贴快捷键**（v3.0的 `Ctrl+Shift+V` 或 v2.0的 `Shift+Insert`），让程序帮你无情地敲进去！

### 方案一：物理外挂（OCR 光学字符识别）

网页代码防得住键盘，但防不住你的眼睛和屏幕。利用 OCR 工具直接从屏幕截图中“抠”出纯文本，这是成功率 100% 的终极解法。

* **推荐工具 1 (微软官方)**：安装 **PowerToys**，使用其内置的**文本提取器**。按下 `Win + Shift + T` 框选屏幕代码，瞬间进入剪贴板。
* **推荐工具 2 (国民应用)**：打开微信 (`Alt + A`) 或 QQ (`Ctrl + Alt + A`) 截图，框选代码后点击工具栏的**“提取文字”**或**“文”**字图标，复制即可。

### 方案二：开发者控制台强取 (F12)

防复制脚本通常作用于鼠标和键盘事件，但页面真实的 DOM 数据依然存在于浏览器内存中。

1. 在限制复制的网页按下 `F12` 打开开发者工具。
2. 切换到 **Console (控制台)** 标签页。
3. 输入 `document.body.innerText` 并回车。
4. 控制台会直接打印出当前网页的所有纯文本。在这里选中你需要的代码进行复制（控制台内没有任何防复制限制）。

### 方案三：无视限制的终极插件 (浏览器扩展)

如果你不想每次都截图或打开控制台，可以安装专门对付这种恶劣环境的扩展插件，通过浏览器内部权限击碎防线。

1. 在 Edge 扩展商店或 Chrome 网上应用店搜索并安装：**`Absolute Enable Right Click & Copy`**（绝对启用右键和复制）。
2. 在被限制的网页上，点击浏览器右上角该插件的图标。
3. 勾选 **“Enable Copy”**（启用复制）。如果依然无法选中文字，再次点击图标，勾选底部的 **“Absolute Mode”**（绝对模式）。
4. 此时网页所有的防选中、防右键机制将被强制摧毁，你可以像平常一样鼠标选中代码并复制。然后切回沙箱，用本工具**强行贴贴**。
