# you_can_copy_and_paste
我是一个大学生，在写水课作业的时候发现网站有复制贴贴的限制，于是开发了这个强制复制贴贴工具，只需要下载双击就能使用，复制是ctrl+shift+c，贴贴是ctrl+shift+v。后续会持续更新它的功能，有然后问题和建议都可以联系我。

# 🪄 You-Can-Copy-And-Paste

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![AHK Version](https://img.shields.io/badge/AutoHotkey-v2.0+-green.svg)

受够了某些网页或文档恶心的“禁止复制/粘贴”限制？**You-Can-Copy-And-Paste** 是一款基于 AutoHotkey v2 开发的底层效率神器。它通过底层键盘指令强制提取受限文本，并在粘贴时模拟真人真实打字节奏，完美绕过前端拦截与行为监控，让你的剪贴板重获自由。

---

## ✨ 核心特性 (Features)

- 💪 **强力复制突破限制**：采用底层 `Ctrl+Insert` 指令提取文本，无视常规前端热键拦截。
- 🧠 **防监控智能粘贴**：
  - **动态延迟注入**：支持“固定延迟”与“随机波动延迟”，模拟人类真实打字速度，安全绕过防作弊检测环境。
  - **自动排版净化**：粘贴前自动裁剪文本首尾空白，智能合并连续多余空行。
- 🎛️ **现代化图形面板 (GUI)**：告别改代码，提供可视化控制台，轻松调节基础延迟、波动毫秒数及开机自启功能。
- 🤫 **无感静音反馈**：摒弃吵闹的系统提示音，采用极简 ToolTip 视觉反馈，2秒自动隐藏，专注不打扰。
- 🛑 **紧急制动开关**：在模拟长文本打字粘贴时，按下 `Esc` 键可瞬间强制中止，防止失控。

## ⌨️ 默认快捷键 (Shortcuts)

| 快捷键 | 功能 | 说明 |
| :--- | :--- | :--- |
| `Ctrl` + `Shift` + `C` | **强力复制** | 请确保选中目标文本后按下 |
| `Ctrl` + `Shift` + `V` | **智能粘贴** | 自动净化排版并按照设定的延迟输出 |
| `Esc` | **紧急停止** | 立即中止正在进行的打字输出并重置程序 |

## 🚀 安装与使用 (Installation & Usage)

### 方案一：普通用户（推荐，免安装环境）
1. 前往本仓库的 [Releases 页面](../../releases/latest) 下载最新版的 `you_can_copy_and_paste.exe`。
2. 双击运行即可，无需安装。
3. 观察任务栏右下角，出现绿色 "H" (或自定义) 托盘图标即代表运行成功。右键图标可进入“设置”面板。

### 方案二：开发者运行源码
1. 请确保您的电脑已安装 [AutoHotkey v2](https://www.autohotkey.com/v2/) 环境。
2. 克隆或下载本仓库源码。
3. 双击运行 `you_can_copy_and_paste.ahk`。

## ⚠️ 杀软误报说明 (Antivirus False Positives)

由于本工具需要突破系统限制，使用了**底层按键模拟**和**全局热键拦截**技术。对于未经高昂数字签名的独立开源软件，极易被 Windows Defender、360 等杀毒软件判定为可疑行为并进行拦截。
- **解决方案**：本软件 100% 开源，绝无任何网络请求和恶意代码。请在杀毒软件中选择**“允许运行”**或将其加入**信任白名单**。
